# API Explorer

A vanilla JS app to learn APIs by calling real public endpoints from the browser. No backend — just open `index.html` and click **Try it**.

## APIs included

- **Random User** — random user + avatar
- **Dog CEO** — random dog image
- **Quotable** — random quote
- **REST Countries** — country search
- **GitHub** — user's public repos
- **Custom** — send your own GET request (any URL)

## Run locally

Open `index.html` in a browser, or run a local server:

```bash
npx serve .
# or: python3 -m http.server 8000
```

All APIs are free and don't require an API key.
